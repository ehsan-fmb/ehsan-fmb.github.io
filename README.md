# Personal Research Website

This is a simple static researcher website designed for GitHub Pages.

## Files

- `index.html` — main webpage content
- `styles.css` — visual design
- `script.js` — small JavaScript for the footer year
- `Ehsan_Futuhi_CV.pdf` — downloadable CV

## How to publish on GitHub Pages

1. Create a GitHub repository named exactly:

   ```text
   your-github-username.github.io
   ```

2. Upload these files to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. Your site should appear at:

   ```text
   https://your-github-username.github.io
   ```

## Things to edit

- Replace the Google Scholar link in `index.html` with your exact Google Scholar profile URL.
- Check whether the GitHub and LinkedIn URLs are correct.
- Add a professional photo if desired, then modify the hero section.
- Add paper links, arXiv links, code links, and PDFs when available.
